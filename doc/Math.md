* atan2(y, x)

* cos(x)
  + Returns the cosine of a x.
* floor(x)
  + Returns the smaller integer greater than or equal to a given x.
* log(x)
  + Returns logarifm of x
* log10(x)

* random()
  * random()
    + Returns random float from 0 to 1.

    + Returns random float from 0 to x.
