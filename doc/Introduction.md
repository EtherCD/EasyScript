# Introduction
**EasyScript** - it is a simple programming language. Designed as an addition to projects. It is a dynamically typed programming language.

**While in early development**

## Syntaxis
> Сomments

`// Simple Comment`

`/* Multiline Comment */`

> Create a mutable variable

`var name = 10`

> Creating an immutable variable, **constant**

`const a = 10`

> Changing a variable

`a = 10`

> Calling Functions

`name(arguments)`

> Standard Functions

* See [Libraries](##Libraries)

> Cycles
>> `while (condition) {}`

>> `do {} while (condition)`

>> `for (var i = 0, i < 10, i = i + 1) {}`

> Создание функцый

```es
func test(key1, key2) {
    print(key1, key2)
}
```
